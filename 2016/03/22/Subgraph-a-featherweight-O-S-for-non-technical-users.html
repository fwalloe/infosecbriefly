Subgraph - a featherweight OS for non-technical users

22 March 2016 on Subgraph, linux, OS, operating system, Debian, news

The Debian-based Subgraph OS, first released in 2015, is a featherweight open-source solution that comes with all the security and privacy options pre-configured.

Key features:
Full Disk Encryption

On by default
Tor

All traffic goes through it
Application sandboxing

Using Containe
System and Kernel Security

Grssecurity+++
Secure Mail Services

With simple PGP key management
